## Detailed Description

PageGrab is a free developer toolkit that extracts design intelligence from any website -- no subscriptions, no external servers, no data leaving your browser.

**What it does**

Five tools in one extension:

1. Extract Text -- Pulls page content as clean Markdown with headings, lists, and links preserved. Ready to paste into your docs, LLM prompts, or CMS.

2. LP Analysis -- Breaks down landing page structure: layout patterns, section hierarchy, CTA placement, and content flow. Useful for studying competitor pages or auditing your own.

3. Design System -- Reverse-engineers a site's design system in one click. Extracts the color palette, typography scale, spacing tokens, and recurring component patterns. Output is structured and ready to use as a starting point for your own system.

4. Pick Component -- Click any element on the page to extract its HTML and computed CSS as a self-contained, reusable component. No manual DevTools inspection required.

5. Full Screenshot -- Captures the entire page as a single image, properly handling fixed headers, sticky navs, and pages of any length.

**Why PageGrab exists**

Dedicated tools already cover parts of this workflow. FireShot captures screenshots. DivMagic copies elements for $49/month. But no single tool combines text extraction, layout analysis, design system reverse-engineering, component picking, and full-page screenshots -- especially not for free.

The design system extraction is the core differentiator. Other tools let you copy what you see. PageGrab tells you *why* it looks that way: the underlying color tokens, type scale ratios, spacing rhythm, and component structure. That is the difference between copying a button and understanding the system that produced it.

**Privacy and performance**

- Manifest V3 -- built on the latest Chrome extension platform
- Zero external requests -- all analysis runs locally in your browser
- No account required -- install and use immediately
- No tracking, no analytics, no data collection

**Built for**

- Frontend developers studying well-built sites
- Designers extracting design tokens from live products
- Marketers analyzing competitor landing pages
- Anyone who needs clean text or screenshots from the web

PageGrab is open source and free. No premium tier, no feature gates, no trial period.
