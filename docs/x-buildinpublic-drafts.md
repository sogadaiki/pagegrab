## Day 1 - "What I'm building" + build-in-public declaration

**Draft:**

I've been spending too many hours inside browser DevTools doing work that should take seconds.

So I built PageGrab -- a Chrome extension that combines 5 tools into one:
- Text Extractor (Markdown output)
- Full Page Screenshot
- Design System Extractor

All local. No server. Building in public.

## Day 3 - "Why I built it" (founder story)

**Draft:**

Every time I wanted to understand how a site's design system worked, I'd open DevTools and start digging through minified CSS manually.

Color values buried in variables. Spacing tokens scattered across files. No fast way to surface the full picture.

That frustration is why PageGrab exists. One click, full design system extracted.
